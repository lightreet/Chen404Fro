刚到清远的时候

排队的时间不算短，但风吹过来的时候，整个人已经提前进入假期。那些小小的期待很适合留在游记里。

漂流后的饭总是格外好吃，像给这趟小旅行盖了一个温热的章。没什么宏大的感想，就是很开心。
